# Google Boooks Search App
Google Books Search App is a react app that allows user to search for books using the Google books API,view books,buy, safe to a database and delete them.
![](img/5.png),![](img/6.png)
# Technologies Used
- Javascript
- React
- Node.js
- Express.js
- MongoDB
- Mongoose
- express NPM Package
- Google Books Api
